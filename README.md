# Automatic Prompt Generator for Designers

## Description
This project generates high-quality AI image prompts based on simple user
inputs such as subject, style, mood, lighting, and color palette.

It demonstrates prompt engineering principles using structured templates
without relying on external LLM APIs.

## Features
- Template-based prompt generation
- Customizable prompt parameters
- Designer-friendly output
- Python console-based application

## How to Run
```bash
python generator.py
